开发环境：
linux Red Hat 4.8.5-3
gcc version 4.8.5 2015062
VScode Version: 1.51.0

PPT上的样例拓扑结构如下：
（u-0,v-1,y-2,x-3,z-4)
{ 	
    {   0,   4,  INF,   12,   INF},
    {   4,   0,   7,   INF,   5},
	{ 	INF,   7,   0,  1,	10},
	{   12,  INF,	1,   0,  2},
	{   INF,  5,	10,   2,  0}
}

代码的输出重定向到out.txt中，包括：
（1）收敛时间
（2）消息的总和
（3）每个节点发送的消息数
（4）每个节点接收的消息数
（5）发送消息的上一个节点
